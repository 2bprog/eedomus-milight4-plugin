Ce plugin eedomus permet de piloter les ampoules connectées à un bridge Mi-Light v4.

## Prérequis 

Un brige Mi-Light v4 installé et configuré.

## Installation

Cliquez sur "Configuration" / "Ajouter ou supprimer un périphérique" / "Store eedomus" / "Milight v4" / "Créer"

## Champs a configurer : 

![Configuration Mi-Light v4](https://raw.githubusercontent.com/2bprog/eedomus-milight4-plugin/master/doc/milight4.jpg)

* IP + Port du bridge V4 : IP adresse et port (sous la forme x.x.x.x:yyyy) de votre brige Mi-Light v4, par défaut le port est 8899 
* Type d'ampoule : Ampoule Blanche ou Ampoule RGBW
* Groupe : Le groupe sur lequel est connecté l'ampoule, vous pouvez egalement sélectionner tous les groupes.
